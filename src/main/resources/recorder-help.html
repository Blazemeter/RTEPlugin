BZM RTE emulator help

Keyboard mapping

Keys Action
Arrows move cursor position
Backspace Remove previous character and move cursor one position back
Delete Remove character at cursor position
Enter Send Enter attention key
F[1..12] Trigger associated F[1..12] attention key
SHIFT + F[1..12] Trigger associated F[13..24] attention key
SHIFT + ESC Send SysReq attention key

Status bar

Indicator Description
row: X / column: Y Row and column where the cursor is positioned
First row and column is 1
/ Shows whether keyboard input is enabled or not
Displayed when mainframe application triggers an alarm
This will blink for 5 seconds and is accompanied by an audio alert
